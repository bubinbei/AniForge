# AniForge Docs Vault

Эта папка является Obsidian vault для управления продуктом AniForge.

## Быстрый старт
1. Открыть `docs` как vault в Obsidian.
2. Начать с `00_Vision/Vision.md` и `01_Roadmap/Phase_1_MVP.md`.
3. Для еженедельных синков использовать `templates/Board_Meeting_Notes.md`.

## Карта разделов
- `00_Vision` — стратегия и монетизация.
- `01_Roadmap` — поэтапный план.
- `02_Product` — персоны, flow, фичи, чеклисты ролей.
- `03_Design` — UI принципы и дизайн-задачи.
- `04_Frontend`, `05_Backend`, `06_AI`, `07_DevOps` — инженерные направления.
- `08_Analytics` — метрики и growth.
- `09_Pitch` — презентационные материалы.
- `10_Progress` — спринты и статус.

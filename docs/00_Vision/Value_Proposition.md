# Value Proposition

## Для пользователя
- Быстрый discovery релевантного аниме.
- Централизованный трекинг статусов и оценок.
- AI-рекомендации с понятной логикой.
- Геймификация, повышающая вовлеченность.

## Для владельца продукта
- Модульная архитектура и предсказуемый roadmap.
- Простая эксплуатация на домашнем сервере.
- Плавный переход от pet-project к SaaS-модели.

## Чем отличаемся
- Не копия существующих каталогов.
- Упор на объяснимые рекомендации + progress loop (XP/achievements).
- Практичный self-hosted DevOps как часть ценности проекта.

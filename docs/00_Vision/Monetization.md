# Monetization

## Модель
- Free: до 10 тайтлов суммарно в списках.
- Premium: без лимитов + advanced filters + расширенная аналитика.

## Этапы
1. Phase 1: hardcoded plan rules, без платежей.
2. Phase 3 (mock): имитация биллинга и lifecycle подписки.
3. Phase 3 (real): Stripe Checkout + webhooks + billing events.

## KPI монетизации
- Free -> Premium conversion.
- Churn rate.
- ARPPU.

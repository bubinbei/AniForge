# AniForge Vision

## Миссия
AniForge помогает фанатам аниме быстро находить релевантные тайтлы, вести трекинг прогресса и получать персональные рекомендации с объяснением причин.

## Принципы продукта
- Простота входа: регистрация за минуты, понятный UX.
- Полезность с первого дня: каталог + личные списки + базовые рекомендации.
- Прозрачность AI: каждое предложение сопровождается объяснением "почему".
- Путь к SaaS: free ценность высокая, premium усиливает power-user сценарии.

## North Star Metric
Доля пользователей, добавивших >=3 тайтлов в лист в первые 7 дней.

## Долгосрочная цель
Вырастить self-hosted-ready SaaS-решение портфельного уровня: продукт, который можно показать как инженерно зрелый стартап.

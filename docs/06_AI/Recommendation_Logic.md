# Recommendation Logic (v1)

## Input signals
- Оценки пользователя (приоритет >=8).
- Частота жанров в высоко оцененных тайтлах.
- Исключение уже добавленных тайтлов.

## Алгоритм
1. Собрать любимые жанры по score.
2. Найти кандидатов с пересечением жанров.
3. Отсортировать по `avgRating` и `popularityScore`.
4. Для каждого кандидата сгенерировать reason string.

## Explainability шаблон
`Рекомендовано: вы часто высоко оцениваете жанры X, Y`.

## Ограничения v1
- Нет embedding/semantic matching.
- Нет персонализации по времени просмотра.

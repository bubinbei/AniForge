# Future ML

## Возможные апгрейды
- Embedding similarity (title + tags + synopsis).
- Learning-to-rank по пользовательским событиям.
- Context-aware recommendations (день недели, настроение, длина тайтла).

## Data prerequisites
- События просмотра и оценок.
- Нормализация жанров/тегов.
- История взаимодействий с рекомендациями.
